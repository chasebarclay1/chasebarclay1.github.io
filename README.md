# Chase Barclay — Portfolio Website

Personal portfolio website for Chase Barclay, Computer Engineering student at Georgia Tech.

## Live Site

Hosted at: `https://chasebarclay1.github.io/`

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build step
- Google Fonts (Inter + JetBrains Mono)
- Canvas-based animated particle background
- Intersection Observer API for scroll animations
- CSS custom properties (glassmorphism design system)

## Features

- Animated particle network background with mouse interaction
- Typing animation cycling through engineering specializations
- Smooth scroll-reveal animations throughout
- Glassmorphism card design
- Responsive layout (mobile-first)
- Counter animations for stats
- Mouse-light hover effect on project cards
- Active nav highlight on scroll

## Hosting on GitHub Pages

1. Push this repository to GitHub under the name `chasebarclay1.github.io`
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder `/`
4. Site will be live at `https://chasebarclay1.github.io`
